# $67 - The Fartcoin of 2026

A crypto landing page with terminal interface and matrix rain effect.

## Features

- Interactive question page (YES/NO)
- ASCII art logo with glitch effect
- Custom cursor animation
- Scanline and glow effects
- Matrix rain background
- Integrated terminal with commands
- Live chart placeholder (DexScreener)
- Responsive design

## Terminal Commands

| Command | Description |
|---------|-------------|
| `67.help` | Show all available commands |
| `67.about` | Learn about $67 |
| `67.mission` | Our mission statement |
| `67.buy` | How to buy $67 |
| `67.ca` | Get contract address |
| `67.why` | Why 67? |
| `67.meme` | Random 67 meme |
| `67.price` | Current vibes |
| `67.whale` | The whale watches (ASCII art + typing effect) |
| `67.lore` | The forbidden knowledge |
| `67.fortune` | Receive your alpha |
| `67.progress` | Project status with progress bars |
| `67.signal` | Ritual scan with whale animation + chart pulse |
| `clear` | Clear terminal |

## Setup

1. Clone the repository
2. Open `index.html` in your browser
3. Or deploy to any static hosting (Vercel, Netlify, GitHub Pages)

## Configuration

### Update Contract Address
Replace `TBA_CONTRACT_ADDRESS` in `js/main.js` with your actual contract address.

### Enable Live Chart
In `index.html`, replace `TOKEN_ADDRESS` in the DexScreener iframe URL with your token's contract address, then show the iframe and hide the placeholder.
